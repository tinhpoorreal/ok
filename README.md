# Cashback RCGV Dashboard

Ứng dụng web tĩnh dựa trên `deep-research-report.md` và `cashback_RCGV_final.xlsx`.

## Tính năng

- Dashboard số lượng thẻ theo danh mục cashback.
- Tìm kiếm và lọc thẻ theo danh mục, ngân hàng, tên thẻ hoặc ghi chú ưu đãi.
- Máy tính ước tính cashback theo chi tiêu tháng, tỷ lệ hoàn và trần hoàn.
- Bảng các thẻ còn thiếu thông tin cần xác minh.

## Chạy app

Mở trực tiếp file `index.html` trong trình duyệt. App không cần backend và không gửi dữ liệu ra ngoài.

## Nguồn dữ liệu

- Báo cáo nghiên cứu: `deep-research-report.md`
- Workbook: `cashback_RCGV_final.xlsx`

App chỉ là bản MVP/dashboard tra cứu, không xử lý thanh toán, không lưu số thẻ đầy đủ và không kết nối ngân hàng thật.
